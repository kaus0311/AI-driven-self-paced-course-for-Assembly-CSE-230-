# AI-driven-self-paced-course-for-Assembly-CSE-230-
This project is about creating an AI-driven, self-paced learning platform for CSE 230 (Assembly Language). It focuses on giving students instant feedback on errors, adaptive mastery checks, and personalized guidance. The system also supports teaching assistants and instructors by providing grading tools, analytics, and course management features. Overall, it aims to make assembly learning more engaging, efficient, and accessible while easing workload for graders and professors.
